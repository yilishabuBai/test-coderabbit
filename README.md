# test-coderabbit

A project designed to test CodeRabbit's AI code review capabilities and boundaries.

## Purpose

This repository serves as a testing ground for evaluating CodeRabbit's ability to:

- Detect common bugs and logic errors
- Identify security vulnerabilities
- Flag performance anti-patterns
- Enforce coding best practices
- Integrate with various linting tools

## Getting Started

```bash
npm install
```
